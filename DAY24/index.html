💻 Code Examples

Insert 5 Products with insertMany()

db.products.insertMany([
  {
    name: "Laptop Pro 15",
    category: "Electronics",
    price: 1299.99,
    stock: 25,
    specs: {
      color: "Silver",
      weight: "1.8kg",
      processor: "Intel i7",
      ram: "16GB"
    }
  },
  {
    name: "Wireless Mouse",
    category: "Electronics",
    price: 29.99,
    stock: 150,
    specs: {
      color: "Black",
      weight: "0.1kg",
      connectivity: "Bluetooth",
      battery: "AA x2"
    }
  },
  // ... 3 more products
])

Find All Electronics

// Query by category
db.products.find({ category: "Electronics" })

// Expected: 2 products
// - Laptop Pro 15
// - Wireless Mouse

Sort by Price & Limit to Top 2

// Sort descending (-1) and limit to 2
db.products.find().sort({ price: -1 }).limit(2)

// Expected:
// 1. Laptop Pro 15 - $1299.99
// 2. Wooden Desk - $399.99

Query Nested Objects (specs)

// Find products by color (nested field)
db.products.find({ "specs.color": "Black" })

// Find by weight (using regex for patterns)
db.products.find({ "specs.weight": /^0\./ })

// Multiple nested conditions
db.products.find({
  "specs.color": "Silver",
  "specs.processor": "Intel i7"
})
