## Pages Overview

### 1. Homepage (index.html)
- **Hero Section**: Conference title, date (March 15-16, 2026), and location
- **Description**: Overview of the conference with 2 paragraphs
- **Key Highlights**: Unordered list with 5 bullet points
- **Quick Stats Table**: Attendees, Speakers, Sessions, and Workshops counts
- **Call-to-Action**: Link to registration page
- **Navigation**: Links to all pages
- **Footer**: Copyright information

### 2. About Page (about.html)
- **Mission & History**: 4 detailed paragraphs about the conference
- **Why Attend**: Unordered list of 7 benefits
- **Target Audience**: Ordered list of 8 professional roles
- **Conference Tracks**: 5 tracks with h3 headings and descriptions
- **Statistics Table**: Historical data including years running, attendees, etc.

### 3. Speakers Page (speakers.html)
- **6 Keynote Speakers**: Each with:
  - Speaker photo (placeholder image reference)
  - Name (h3 heading)
  - Job title and company
  - 2-3 paragraph biography
  - List of expertise areas
- Organized using divs for clear separation

### 4. Schedule Page (schedule.html)
- **Day 1 Table**: March 15, 2026 with 14 time slots
- **Day 2 Table**: March 16, 2026 with 14 time slots
- Each table includes: Time, Session Title, Speaker, Room
- Properly structured with `<thead>` and `<tbody>`
- Includes sessions, breaks, lunch, and networking events

### 5. Sponsors Page (sponsors.html)
- **Platinum Sponsors**: 2 sponsors with logos and descriptions
- **Gold Sponsors**: 3 sponsors with logos and descriptions
- **Silver Sponsors**: 4 sponsors with logos and descriptions
- **Sponsor Benefits Table**: Comparison of benefits by tier
- **Become a Sponsor Section**: Contact information and CTA

## Technical Implementation

### HTML5 Standards
- ✓ Proper DOCTYPE declaration on all pages
- ✓ Semantic HTML structure (header, nav, section, footer)
- ✓ Consistent `<head>` section with charset, viewport, description, and title
- ✓ Proper heading hierarchy (h1, h2, h3)

### Navigation
- ✓ Identical navigation menu on all pages
- ✓ Working relative links between pages
- ✓ Home, About, Speakers, Schedule, Sponsors links

### Content Elements
- ✓ Tables with proper structure (`<table>`, `<tr>`, `<th>`, `<td>`, `<thead>`, `<tbody>`)
- ✓ Both ordered and unordered lists
- ✓ Divs for content grouping
- ✓ Alt attributes on all images
- ✓ Properly indented and formatted code

### Images
The `images/` folder contains placeholders for:
- 6 speaker photos (speaker1.jpg through speaker6.jpg)
- 9 sponsor logos (various PNG files for platinum, gold, and silver tiers)

**Note**: The actual image files need to be added. You can use placeholder services like:
- https://via.placeholder.com/200x200 (for speaker photos)
- https://via.placeholder.com/300x100 (for sponsor logos)

## How to Use

1. **View the Website**:
   - Open `index.html` in any web browser
   - Navigate through pages using the menu

2. **Add Images**:
   - Replace placeholder image references with actual images
   - Place image files in the `images/` folder
   - Recommended dimensions:
     - Speaker photos: 200x200 pixels
     - Sponsor logos: 200-300 pixels wide

3. **Customize Content**:
   - All content is easily editable in the HTML files
   - Maintain the existing structure for consistency
   - Update dates, names, and descriptions as needed
