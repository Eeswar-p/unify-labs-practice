⚡ Quick Facts

  • updateOne() - Affects FIRST matching document
  • updateMany() - Affects ALL matching documents
  • deleteMany() - DELETE is final and irreversible!
  • Always find() first - Preview before deleting
  • Combine operators - Use multiple in one command

⚙️ Update Operators Explained

1️⃣ $inc - Increment (For Numbers)

Increases a numeric field by specified value

Syntax

{ $inc: { field: number } }

Example

// Increase Electronics price by 10
db.products.updateMany(
  { category: "Electronics" },
  { $inc: { price: 10 } }
)

Use Cases

  • Adjust prices up/down
  • Update stock levels
  • Increment counters (views, sales)
  • Track metrics

Key Points

  • Works with negative numbers: { $inc: { stock: -5 } }
  • Can increment multiple fields at once
  • Creates field if it doesn't exist

2️⃣ $set - Set Value (For Anything)

Sets a field to a specific value (creates if not exists)

Syntax

{ $set: { field: value } }

Example

// Mark expensive items as featured
db.products.updateMany(
  { price: { $gt: 500 } },
  { $set: { featured: true } }
)

Use Cases

  • Add/update boolean flags
  • Change categories
  • Set descriptions
  • Update metadata

Key Points

  • Works with any data type
  • Can set nested fields: { $set: { "specs.color": "Blue" } }
  • Set multiple fields: { $set: { field1: val1, field2: val2 } }

3️⃣ $push - Add to Array

Adds an item to an array field (creates if not exists)

Syntax

{ $push: { arrayField: value } }
{ $push: { arrayField: { $each: [val1, val2] } } }

Example

// Add single tag
db.products.updateOne(
  { name: "Wireless Mouse" },
  { $push: { tags: "new-arrival" } }
)

// Add multiple tags
db.products.updateOne(
  { name: "Laptop Pro 15" },
  { $push: { tags: { $each: ["hot-deal", "premium"] } } }
)

Use Cases

  • Add reviews/ratings
  • Track history/activity
  • Manage tags
  • Append comments

Key Points

  • $each adds multiple items
  • Creates array if doesn't exist
  • Can add objects: { $push: { reviews: { userId: "u1", rating: 5 } } }

4️⃣ deleteMany() - Bulk Delete

Deletes ALL documents matching filter

Syntax

db.collection.deleteMany({ filter })

Example

// Delete all zero stock items
db.products.deleteMany({ stock: 0 })

// Verify first
db.products.find({ stock: 0 })  // See what you're deleting
db.products.deleteMany({ stock: 0 })  // Then delete

Use Cases

  • Remove discontinued products
  • Clean up test data
  • Archive old records
  • Bulk clean up

⚠️ Safety Points

  • ALWAYS find() first to preview
  • DELETE IS FINAL - No undo!
  • Use specific filters - Never deleteMany({})!
  • Check countDocuments() before/after

💻 Complete Code Examples

Example 1: Increase Electronics Price (+$10)

MongoDB Shell (mongosh)
db.products.updateMany(
  { category: "Electronics" },
  { $inc: { price: 10 } }
)
Node.js
const result = await collection.updateMany(
  { category: "Electronics" },
  { $inc: { price: 10 } }
);
console.log(`Modified: ${result.modifiedCount}`);
Expected Result:
  • Laptop Pro 15: $1299.99 → $1309.99
  • Wireless Mouse: $29.99 → $39.99

Example 2: Mark Expensive Items (> $500)

)
Node.js
const result = await collection.updateMany(

  { $set: { featured: true } }
);
console.log(`Modified: ${result.modifiedCount}`);
Expected Result:
  • Only Laptop Pro 15 (now $1309.99) marked featured: true

MongoDB Shell - Single Tag

MongoDB Shell - Multiple Tags

Example 4: Delete Zero Stock Items

MongoDB Shell - Safety First
// Step 1: Preview
db.products.find({ stock: 0 })

// Step 2: Count
db.products.countDocuments({ stock: 0 })

// Step 3: Delete
db.products.deleteMany({ stock: 0 })

// Step 4: Verify
db.products.countDocuments()
Node.js
// Count before
const countBefore = await collection.countDocuments({ stock: 0 });
console.log(`Items to delete: ${countBefore}`);

// Delete
const result = await collection.deleteMany({ stock: 0 });
console.log(`Deleted: ${result.deletedCount}`);

// Count after
const countAfter = await collection.countDocuments({ stock: 0 });
console.log(`Zero stock now: ${countAfter}`);
Expected Result:
  • Test item with stock: 0 deleted
  • countDocuments({ stock: 0 }) returns 0

Example 5: Combine Multiple Operators

MongoDB Shell
// Update one item with multiple operators
db.products.updateOne(
  { name: "Laptop Pro 15" },
  {
    $inc: { price: 50, stock: -2 },
    $set: { featured: true, lastUpdated: new Date() },
    $push: { tags: "updated" }
  }
)
Node.js
const result = await collection.updateOne(
  { name: "Laptop Pro 15" },
  {
    $inc: { price: 50, stock: -2 },
    $set: { featured: true, lastUpdated: new Date() },
    $push: { tags: "updated" }
  }
);
console.log(`Modified: ${result.modifiedCount}`);
Expected Result:
  • Price: $1309.99 + 50 = $1359.99
  • Stock: 15 - 2 = 13
  • featured: true
  • tags now includes "updated"
