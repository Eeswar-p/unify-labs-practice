💻 API & async/await Reference

🔗 fetch() API 

// Make HTTP request
const response = await fetch(url);

// Check if successful
if (!response.ok) {
    throw new Error(`HTTP ${response.status}`);
}

// Parse response
const data = await response.json();

// Use data
console.log(data);

Fetch data from URLs. Returns a Promise that resolves to a Response object.

⏳ async/await 

// Define async function
async function fetchData() {
    // await pauses execution
    const response = await fetch(url);
    const data = await response.json();
    
    // Returns a Promise
    return data;
}

// Call async function
fetchData();

Async functions return Promises. Use await to pause execution until Promise resolves.

⚠️ Error Handling 

// Handle errors with try/catch
async function fetchData() {
    try {
        const response = await fetch(url);
        if (!response.ok) throw new Error('Failed');
        const data = await response.json();
        return data;
    } catch (error) {
        console.error('Error:', error.message);
        // Handle error gracefully
    }
}

Try/catch blocks handle errors from Promises and async functions.

📦 Response Methods 

// Different response types
const jsonData = await response.json();
const textData = await response.text();
const blobData = await response.blob();
const arrayData = await response.arrayBuffer();

// Response properties
response.status         // HTTP status code
response.ok             // true if 200-299
response.statusText     // "OK", "Not Found", etc.
response.headers        // Response headers

Different methods to parse and access response data and metadata.

🎁 Promise Methods 

// Wait for multiple Promises
const results = await Promise.all([
    fetch(url1).then(r => r.json()),
    fetch(url2).then(r => r.json())
]);

// Wait for first to complete
const firstDone = await Promise.race([
    fetch(url1),
    fetch(url2)
]);

// Handle all, even if some fail
const all = await Promise.allSettled([
    fetch(url1),
    fetch(url2)
]);

Combine multiple Promises with Promise.all(), race(), and allSettled().

💡 Common Patterns

Basic Fetch

async function getData() { const res = await fetch(url); if (!res.ok) throw Error(res.status); return res.json(); }

With Loading State

async function fetchAndDisplay() { loader.hidden = false; try { const data = await fetch(url) .then(r => r.json()); render(data); } finally { loader.hidden = true; } }

Error Handling

async function safeFetch(url) { try { const res = await fetch(url); if (!res.ok) { throw Error(`${res.status}`); } return await res.json(); } catch (e) { console.error(e); showError(e.message); } }

Parallel Requests

// Fetch multiple APIs at once const [users, posts] = await Promise.all([ fetch(usersUrl).then(r => r.json()), fetch(postsUrl).then(r => r.json()) ]); // Much faster than sequential

Search & Filter

// Filter data client-side const filtered = data.filter(item => item.title.toLowerCase() .includes(query.toLowerCase()) ); // More efficient for small datasets

Retry Logic

async function fetchWithRetry(url, retries = 3) { try { return await fetch(url).then(r => r.json()); } catch (e) { if (retries > 0) { return fetchWithRetry( url, retries - 1); } throw e; } }
