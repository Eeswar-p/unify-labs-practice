💻 DOM Methods Reference

🔍 querySelector 

// Select element by CSS selector
const element = document.querySelector(selector);

// Examples
const input = document.querySelector('#taskInput');
const list = document.querySelector('.task-list');
const items = document.querySelector('li.task-item');

// Returns: First matching element
// Returns: null if not found

Selects the FIRST element matching a CSS selector. Most flexible method for selection.

🔍 querySelectorAll 

// Select all matching elements
const elements = document.querySelectorAll(selector);

// Examples
const allItems = document.querySelectorAll('.task-item');
const btns = document.querySelectorAll('button');

// Returns: NodeList of all matches
// Returns: Empty NodeList if none found

// Iterate over results
allItems.forEach(item => {
    console.log(item);
});

Selects ALL elements matching a CSS selector. Returns a NodeList you can loop through.

🏷️ getElementById 

// Select element by ID
const element = document.getElementById(id);

// Examples
const input = document.getElementById('taskInput');
const btn = document.getElementById('addBtn');

// Returns: Element with matching ID
// Returns: null if not found

// Quick & fast (best for unique IDs)
const taskList = document.getElementById('taskList');

Selects element by unique ID. Fastest method for selecting by ID (more specific than querySelector).

⚡ addEventListener 

// Add event listener
element.addEventListener(event, handler);

// Examples
const btn = document.querySelector('#addBtn');
btn.addEventListener('click', () => {
    console.log('Button clicked!');
});

const input = document.querySelector('#taskInput');
input.addEventListener('keypress', (e) => {
    if (e.key === 'Enter') {
        addTask();
    }
});

Attaches an event listener to an element. Called when specified event occurs.

✨ Create & Append 

// Create new element
const newElement = document.createElement(tag);

// Set properties
newElement.textContent = 'Hello';
newElement.className = 'task-item';
newElement.id = 'task-1';

// Add to DOM
const container = document.querySelector('#taskList');
container.appendChild(newElement);

// Add multiple children
newElement.innerHTML = `Task
`;

Create new DOM elements dynamically and add them to the page.

🎨 classList Methods 

// Toggle class
element.classList.toggle('completed');

// Add class
element.classList.add('active');

// Remove class
element.classList.remove('active');

// Check if has class
if (element.classList.contains('completed')) {
    console.log('Task is completed');
}

// Replace class
element.classList.replace('active', 'inactive');

Manipulate CSS classes on elements. Perfect for toggling styles dynamically.

🗑️ Remove Methods 

// Remove element (modern)
element.remove();

// Remove child (legacy)
parent.removeChild(child);

// Examples
const item = document.querySelector('#task-1');
item.remove(); // Removes from DOM

// Or with parent
const list = document.querySelector('#taskList');
const item = list.querySelector('.task-item');
list.removeChild(item);

Remove elements from the DOM. Use remove() for modern browsers (simpler).

💡 Common Patterns

Toggle Completion

// Click to toggle 'completed' class taskItem.addEventListener('click', () => { taskItem.classList.toggle('completed'); });

Simple pattern for toggling CSS classes on elements.

Delete Item

// Delete button removes item from DOM deleteBtn.addEventListener('click', (e) => { e.stopPropagation(); taskItem.remove(); });

Use e.stopPropagation() to prevent event bubbling.

Create List Item

// Create task item with Delete button const li = document.createElement('li'); li.className = 'task-item'; li.innerHTML = `${text} `;

Combine createElement and innerHTML for complex elements.

Event Delegation

// One listener on parent catches child clicks list.addEventListener('click', (e) => { if (e.target.classList.contains('delete-btn')) { e.target.parentElement.remove(); } });

Attach one listener to parent instead of many to children.

Keyboard Events

// Detect Enter key for form submission input.addEventListener('keypress', (e) => { if (e.key === 'Enter') { addTask(); } });

Handle keyboard input for better user experience.

Update UI State

// Update statistics after each change function updateStats() { const total = document.querySelectorAll('.task-item').length; const completed = document.querySelectorAll('.task-item.completed').length; document.getElementById('totalCount').textContent = total; }

Keep UI in sync with data by updating after changes.
