Semantic Audit

Comprehensive checklist for HTML semantic standards and accessibility compliance

✓ Image Alt Attributes

Every image must have a meaningful <alt> attribute that describes its content for screen readers.

<img src="hero.jpg" alt="Team collaboration meeting in modern office">

✓ Heading Structure

Heading hierarchy should never skip levels (H1 → H2 → H3, never H1 → H3).

<h1>Page Title</h1>
<h2>Section</h2>
<h3>Subsection</h3>

✓ ARIA Labels for Icons

Icon-only buttons must have aria-label to describe their purpose.

<button aria-label="Close menu">
  <svg>...</svg>
</button>

CSS Architecture

OOCSS methodology with CSS variables for maintainable, scalable stylesheets

OOCSS Principles

Separate Structure & Skin

Keep layout (structure) separate from styling (skin) using reusable classes.

/* Structure */
.button { padding: 12px 24px; }

/* Skin */
.button--primary { bg: var(--color-primary); }

Separate Container & Content

Styles should not depend on where elements are located in the DOM.

/* Bad */
.header .button { color: white; }

/* Good */
.button { color: white; }

Performance Optimization

Essential techniques for animation and rendering performance

🚀 Optimal Animation Properties

Animate only transform and opacity for 60fps performance

Property Performance Usage
transform ✓ Excellent Position, scale, rotate
opacity ✓ Excellent Fade in/out
width, height ✗ Poor Triggers layout reflow
left, top ✗ Poor Triggers layout reflow

⚡ will-change Property

Hint to browser about upcoming animations on an element

/* Before animation */
.element { will-change: transform, opacity; }

/* After animation ends */
.element { will-change: auto; }
Note: Use sparingly. Overuse causes memory overhead.

📊 GPU Acceleration

Create new rendering layers for complex animations

.animated {
  transform: translateZ(0);
  /* Creates new stacking context */
  backface-visibility: hidden;
  /* Prevent flickering */
}
