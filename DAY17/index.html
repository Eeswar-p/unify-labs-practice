📚 Array Methods Explained

🔍 filter() Method

// Filter completed tasks
const completed = tasks.filter(
  task => task.status === 'completed'
);

// Creates NEW array with items passing test
// Original array unchanged

Returns new array with elements that pass the test function.

🗺️ map() Method

// Add tax to all prices
const withTax = prices.map(
  price => price * (1 + taxRate)
);

// Transforms each element
// Returns NEW array same length

Transforms each element and returns new array of same length.

📊 reduce() Method

// Sum all expenses
const total = expenses.reduce(
  (sum, expense) => sum + expense.amount,
  0  // Initial value
);

// Reduces array to single value

Reduces array to a single value using accumulator function.

⛓️ Chaining Array Methods

// Filter → Map → Reduce (all together!)
const total = orders
  .filter(order => order.status === 'shipped')    // Keep only shipped
  .map(order => order.price * 1.1)                // Add 10% tax
  .reduce((sum, price) => sum + price, 0);        // Calculate total

// Powerful combination for data processing!
