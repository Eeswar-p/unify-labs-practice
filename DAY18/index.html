💻 Pet Class Implementation

🏗️ Class Definition 

class Pet {
    constructor(name, type) {
        this.name = name;
        this.type = type;
        this._health = 100;
    }
}

The Pet class has a constructor that initializes name, type, and private _health property (starting at 100).

📥 Health Getter 

get health() {
    return this._health;
}

// Usage
console.log(pet.health); // 100

Getters allow you to access private properties like regular properties (no parentheses needed).

📤 Health Setter 

set health(value) {
    if (value > 100) {
        this._health = 100;
    } else if (value < 0) {
        this._health = 0;
    } else {
        this._health = value;
    }
}

Setters validate and constrain values. Here, health is clamped between 0-100 automatically.

⚙️ Pet Methods 

feed() {
    this.health += 10;
    return `${this.name} has been fed!`;
}

play() {
    this.health -= 15;
    return `${this.name} is playing!`;
}

getStatus() {
    return `${this.name} (${this.type})
    Health: ${this.health}/100`;
}

Methods modify health using the setter, which ensures values stay within valid range.

🔒 Encapsulation 

// Private property (by convention)
this._health = 100;

// Public interface
get health() { ... }
set health(value) { ... }

// Direct access bypasses validation
pet._health = 150; // ❌ Bad
pet.health = 150;  // ✅ Good (clamped to 100)

Use underscore prefix for private properties. Access via getters/setters for validation.

🎯 Complete Example 

// Create pet
const myPet = new Pet('Fluffy', '🐶 Dog');

// Feed pet (increases health)
myPet.feed(); // Health: 100 (capped)

// Play with pet
myPet.play(); // Health: 85

// Check status
console.log(myPet.getStatus());
// Output: Fluffy (🐶 Dog)
//         Health: 85/100
