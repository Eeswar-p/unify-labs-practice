A while loop keeps repeating as long as a condition is true. In our game, we keep asking for guesses while the user hasn't guessed correctly and has attempts remaining.

🔍 Binary Search Algorithm

The optimal strategy for guessing is binary search - always guess the middle number to eliminate half the remaining possibilities. 

// Always guess the middle
let low = 1, high = 100;
while (low <= high) {
    let mid = Math.floor(
        (low + high) / 2
    );
}

⚠️ Common Mistakes

❌ Infinite Loop

Forgetting to update the loop condition leads to infinite loops!

// ❌ WRONG - Creates infinite loop
let i = 0;
while (i < 10) {
    console.log(i);
    // i++ is missing!
}
// ✅ CORRECT - Increments i
let i = 0;
while (i < 10) {
    console.log(i);
    i++;  // Update condition!
}

❌ Type Confusion

"50" (string) is different from 50 (number) in comparisons!

// ❌ WRONG - String vs Number
let guess = "50";  // String from input
if (guess === 50) {
    // Never true!
}
// ✅ CORRECT - Convert to number
let guess = parseInt("50");
if (guess === 50) {
    // Now true!
}

❌ Wrong Condition

Using OR (||) instead of AND (&&) makes the loop run when it shouldn't!

// ❌ WRONG - Uses OR
while (guess !== secret || 
       attempts > 0) {
    // Runs even when out of attempts!
}
// ✅ CORRECT - Uses AND
while (guess !== secret && 
       attempts > 0) {
    // Stops when either condition fails
}
