📚 How It Works: Arrow Functions

1️⃣ Title Case Function

const toTitleCase = (str) => {
  return str
    .trim()
    .split(' ')
    .filter(word => word.length > 0)
    .map(word => 
      word[0].toUpperCase() + 
      word.slice(1).toLowerCase()
    )
    .join(' ');
};

Uses trim(), split(), map(), and join() to transform text.

2️⃣ Vowel Counter Function

const countVowels = (str) => {
  const vowels = 'aeiouAEIOU';
  return str
    .split('')
    .filter(char => vowels.includes(char))
    .length;
};

Uses split(), filter(), and includes() to count vowels.

3️⃣ Secret Message Function

const censorWords = (str, words) => {
  let result = str;
  words.forEach(word => {
    const regex = new RegExp(
      word.trim(), 
      'gi'
    );
    result = result.replace(regex, '***');
  });
  return result;
};

Uses RegExp, replace(), and forEach() for censoring.
