Variables Explained

Three ways to declare variables in JavaScript

const

Recommended

Block-scoped, cannot be reassigned after initialization

const age = 25;
  • ✅ Prevents accidental reassignment
  • ✅ Block-scoped (safer)
  • ✅ Must be initialized
  • ✅ Best for most cases

let

For Loops

Block-scoped, can be reassigned but not redeclared

let counter = 0;
counter = 1;
  • ✅ Can be reassigned
  • ✅ Block-scoped
  • ✅ Use in loops
  • ⚠️ Use when value changes

var

Legacy

Function-scoped, can be reassigned and redeclared

var name = "John";
name = "Jane";
  • ⚠️ Hoisting confusion
  • ⚠️ Function-scoped (less safe)
  • ⚠️ Can cause bugs
  • ❌ Avoid in modern code

Arithmetic Operations

Perform calculations with variables

Interactive Math Calculator

How It Works

const num1 = 15;
const num2 = 7;

// Addition
const sum = num1 + num2;  // 22

// Subtraction
const difference = num1 - num2;  // 8

// Multiplication
const product = num1 * num2;  // 105

// Division
const quotient = num1 / num2;  // 2.14...

// Remainder (Modulo)
const remainder = num1 % num2;  // 1

// Exponentiation
const power = num1 ** 2;  // 225

➕ Addition (+)

Adds two numbers together

15 + 7 = 22

➖ Subtraction (-)

Subtracts second from first

15 - 7 = 8

✖️ Multiplication (*)

Multiplies two numbers

15 * 7 = 105

➗ Division (/)

Divides first by second

15 / 7 ≈ 2.14

🔢 Remainder (%)

Returns remainder after division

15 % 7 = 1

⚡ Exponent (**)

Raises to power

2 ** 3 = 8

String Manipulation

Combine strings with concatenation and template literals

Welcome Message Builder

Methods Demonstrated

const name = "Alice";
const role = "Developer";

// Method 1: Concatenation (+)
const msg1 = "Welcome, " + name + "!";

// Method 2: Template Literals (backticks)
const msg2 = `Hi ${name}!`;

// Method 3: Multiple variables
const msg3 = 
  `${name} is a ${role}`;

// Method 4: With expressions
const msg4 = 
  `Age: ${2024 - 2000}`;

Concatenation (+)

Combines strings with + operator

"Hello" + " " + "World"

Result: "Hello World"

⚠️ Can be messy with many strings

Template Literals

Uses backticks and ${} for interpolation

`Hello ${name}`

Result: "Hello Alice"

✅ Preferred modern approach

.concat()

Method to join multiple strings

"Hello".concat(" ", name)

Result: "Hello Alice"

✔️ Clean but less common

With Expressions

Execute code inside template literals

`Total: ${10 + 5}`

Result: "Total: 15"

✅ Very powerful feature

Real-World Examples

Practical applications of variables and math

🛒 Shopping Cart

Calculate total with tax and discount

const price = 100;
const quantity = 3;
const taxRate = 0.08;

const subtotal = price * quantity;
const tax = subtotal * taxRate;
const total = subtotal + tax;

// Result: $324

🎂 Age Calculator

Calculate age from birth year

const birthYear = 2000;
const currentYear = 2024;

const age = currentYear - birthYear;
const msg = 
  `You are ${age} years old`;

// Result: "You are 24 years old"

🌡️ Temperature Conversion

Convert Celsius to Fahrenheit

const celsius = 25;

const fahrenheit = 
  (celsius * 9 / 5) + 32;

const msg = 
  `${celsius}°C = ${fahrenheit}°F`;

// Result: "25°C = 77°F"

📊 Grade Average

Calculate average from test scores

const score1 = 85;
const score2 = 92;
const score3 = 78;

const average = 
  (score1 + score2 + score3) / 3;

// Result: 85 (average)

📏 Distance Calculator

Calculate distance with speed and time

const speed = 60; // km/h
const time = 2.5; // hours

const distance = speed * time;

const msg = 
  `Distance: ${distance} km`;

// Result: "Distance: 150 km"

✍️ Name Formatter

Format names and create personalized messages

const firstName = "John";
const lastName = "Doe";

const fullName = 
  `${firstName} ${lastName}`;

const greeting = 
  `Hello, ${fullName}!`;

// Result: "Hello, John Doe!"
